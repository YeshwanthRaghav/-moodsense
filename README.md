## Quickstart — Reproduce Paper Results

1. Open [Google Colab](https://colab.research.google.com) (free tier, CPU is fine)
2. Upload `moodsense_evaluation.py` or paste it into a code cell
3. Click **Runtime → Run all**
4. Copy the printed TABLE II and TABLE III output into your paper

The script will automatically download:
- `glue/sst2` validation set (872 sentences)
- `dair-ai/emotion` test set (1,000 tweets)
- `distilbert-base-uncased-finetuned-sst-2-english` (Hugging Face)
- `j-hartmann/emotion-english-distilroberta-base` (Hugging Face)

## User Study (Section IV-D)

**Ethics first:** Obtain IRB/ethics board approval before recruiting participants.

1. Create a Google Form with these fields:
   - `participant_id`, `age`, `gender`, `phq2_score` (0–6)
   - `journal_text_1` … `journal_text_5`
   - `helpfulness_rating` (1–5 Likert)
   - `mood_before`, `mood_after` (1–10)

2. Export responses as `user_study_data.csv`

3. Run in Colab (after running `moodsense_evaluation.py` first):
   ```bash
   python user_study_analysis.py
   ```

4. Copy the printed values into Section IV-D of the paper

> ⚠️ **Do not commit `user_study_data.csv` to this repository.** It contains identifiable participant data. Add it to `.gitignore`.

## System Overview

MoodSense combines three sentiment models into a single **Wellbeing Score (0–100)**:

| Model | Role | Weight |
|---|---|---|
| DistilBERT (SST-2) | Binary sentiment confidence | 0.50 |
| VADER | Lexicon-based compound score | 0.30 |
| TextBlob | Polarity score | 0.20 |

**W = (S_bert × 0.50 + S_vader × 0.30 + S_blob × 0.20) × 100**

## Key Results (from paper)

| Model | SST-2 Acc | Emotion F1 | WB MAE | Spearman r |
|---|---|---|---|---|
| VADER only | 0.669 | 0.224 | 19.7 | 0.569 |
| TextBlob only | 0.628 | — | — | — |
| DistilBERT only | 0.911 | 0.895 | 18.3 | 0.566 |
| **MoodSense Ensemble** | **0.906** | **0.602** | **18.3** | **0.566** |
